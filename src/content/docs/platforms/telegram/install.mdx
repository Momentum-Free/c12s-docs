## 1) Adding the c12s Bot to your Telegram Group
To add c12s bot to your group, you need to search in telegram for the c12s bot, open the bot profile, press "Add to Group" and select the group you wish to add the bot to.

---
## 2) Making sure the bot has the correct permissions

When adding the bot to your group, by defaul it will have the following permissions activated:

| Name | Description |
| ---- | ----------- |
| `Admin Rights` | Blank |
| `Change Group Info` | Blank |
| `Ban Users` | Blank |
| `Invite users via Link` | Blank |
| `Pin Messages` | Blank |

<Badge text="Note:" variant="tip" /> To ensure that the bot functions correctly, please avoid changing the default permissions.
